## 基础点灯项目：Bare_LED

这是我的第一个 STM32F103C8T6 纯裸机指针控制点灯项目。抛弃了臃肿的 HAL 库，直接直面硅片物理地址。

## 硬件连线与引脚定义
外设控制：GPIOA
物理引脚：Pin 5 (PA5)
硬件连接：LED正极(长脚) -> PA5，负极(短脚) -> GND

## 底层寄存器物理地址对照表
根据芯片官方参考手册（RM0008）计算得出的硬核物理地址：

 寄存器名称 | 作用说明 | 基地址 | 偏移量 (Offset) | 最终物理地址 |

|RCC_APB2ENR*| 开启 GPIOA 的外设时钟供电 | `0x40021000` | `0x18` | `0x40021018` |
| PIOA_CRL 配置引脚模式（通用推挽输出, 50MHz）| `0x40010800` | `0x00` | `0x40010800` |
| GPIOA_ODR| 控制引脚高低电平切换（3.3V / 0V） | `0x40010800` | `0x0C` | `0x4001080C` |

## 调参日志
* 2026-07-11：
遇到烧录后灯不亮。软件代码逻辑 100% 正确，但芯片死活不跑。
最终排查发现： Reset 复位按键卡死没有弹起！ 手动按压复位后，PA5 成功输出高低电平，外设 LED 完美按照 S1s 节奏闪烁！
* 关于延时：目前使用纯软件 `delay(1000000)` 耗时。由于空循环极度消耗 CPU 资源，未来计划更换为 SysTick 滴答定时器来实现微秒级精准控时。
